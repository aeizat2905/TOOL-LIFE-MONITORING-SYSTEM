🔥 Firebase Setup Required

This app uses Firebase Realtime Database to sync data across all devices in real time. You only need to do this setup once — paste your Firebase config below and it will be saved for future sessions.

1
Go to console.firebase.google.com → Create a project (e.g. mold-tool-life)
2
In your project → click Realtime Database in the left sidebar → Create Database → choose a region → Start in Test mode
3
Click the ⚙ gear icon → Project Settings → scroll to "Your apps" → click </> Web icon → Register app → copy the firebaseConfig object
4
Paste the entire firebaseConfig = { ... } block below (or just the JSON object inside it)
⚠ Invalid config — make sure it includes databaseURL and apiKey

Security note: After setup, go to Firebase Console → Realtime Database → Rules and update to restrict access.
